Данный проект создан с целью предоставления возможности всем желающим бесплатно устраивать расклады Таро.

Установка:
1. С помощью кнопки скачать код с гита качается архив весь
2. Далее в архиве запускается файл setup.exe и по усолчанию прога идет в програм файлс 86 в папку tarot
3. Открывается эта папка и вручную сохдается каталог models и в нее суется файл нейронки из архива или по ссылке (смотря примет ли гитхаб архив с ии)
4. Запускается старт вбс и октрывается сайт
